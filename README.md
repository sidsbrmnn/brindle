# brindle

Front-end development of Brindle, an events & meeting app.

Design inspired from [Landing page - Meet brindle by Outcrowd](https://dribbble.com/shots/6520524-Landing-page-Meet-brindle)

## Build

```cmd
npm i
npm run build
npm start
```
